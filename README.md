# orujibru

Leitor de EPUB com TTS local via kokoro-onnx.

## Uso

1. Instale o kokoro-onnx via `pip3 install kokoro-onnx`.
2. Baixe o modelo `.onnx` e o arquivo de vozes (`voices*.bin`).
3. Inicie o servidor local:

```bash
npm start
```

4. Abra `http://127.0.0.1:5178` no navegador.
5. Preencha os caminhos do modelo e do arquivo de vozes na interface.
